## 4. Pilot Cadence

| Week | Outcome |
|---|---|
| 1 | Registry loaded; golden eval set confirmed. |
| 2 | Hypotheses and evidence sources captured. |
| 3 | Scoring engine and dashboard draft. |
| 4 | Owner review and approval workflow test. |
| 5 | Leadership scorecard pilot. |
| 6 | Go/no-go, fix/defer/scale recommendations. |

## 5. Pilot Exit Criteria

- 100% pilot use cases scored or explicitly marked NotReady/Unproven.
- No unsupported value claims.
- Approval workflow tested successfully.
- At least one real build/defer/fix/scale decision supported.
- Owners agree the output is actionable.
