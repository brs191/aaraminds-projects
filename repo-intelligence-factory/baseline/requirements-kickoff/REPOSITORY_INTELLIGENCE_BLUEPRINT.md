# Repository Intelligence Blueprint

## Graph Schema
Entities:
- Repository
- Module
- File
- Class
- Function
- API Endpoint
- Database Table

Relationships:
- CALLS
- IMPORTS
- DEPENDS_ON

## GraphRAG Strategy
- Static analysis creates facts
- Graph stores relationships
- Vector search provides semantic context
- Agents perform reasoning
