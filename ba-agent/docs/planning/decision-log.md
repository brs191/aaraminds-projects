## Decision status legend

| Status      | Meaning                                                                                                |
| ----------- | ------------------------------------------------------------------------------------------------------ |
| Open        | Decision is defined but not yet resolved at the required gate.                                         |
| Closed      | Decision is accepted for the current baseline.                                                         |
| Conditional | Build may proceed under the stated constraint; the decision must be revisited before the blocked gate. |
| Deferred    | Not required for the immediate thin-slice build; must be decided before later gated work.              |
