# 99 · Archive

Superseded versions live here, dated. Keep this folder lean — only artifacts worth keeping for an audit trail or for diffing against a current version.

## Conventions

- Append a date suffix when archiving: `Aara_Teams_Aggregator_PRD_2026-05-26.docx`
- For very old versions, group by month: `2026-05/`, `2026-06/`, etc.
- Don't archive PowerPoint exports if you still have the editable `.drawio` source — regenerate when needed instead.
